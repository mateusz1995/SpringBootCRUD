## SpringBootCRUD

WEB application enabling CRUD operations on the table containing countries.

## Used technologies:

- Spring Tool Suite 4.1.1
- Spring Boot
- Bootstrap
- Thymeleaf
- Java
- JPA
- H2 embedded database
- Javascript
- jQuery

## How to start:

1. Clone the repository.
2. Run application.
3. Open the browser and paste: http://localhost:8080/
